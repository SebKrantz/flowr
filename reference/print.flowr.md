# Print Flowr Object

Print method for objects of class `flowr`. Displays a summary of the
flow assignment result.

## Usage

``` r
# S3 method for class 'flowr'
print(x, ...)
```

## Arguments

- x:

  An object of class `flowr`, typically returned by
  [`run_assignment`](https://sebkrantz.github.io/flowr/reference/run_assignment.md).

- ...:

  Additional arguments (currently ignored).
